Установка шрифтов OpenType PS.

- Шрифты OpenType на основе PostScript (OT/PS) состоят из одного файла с расширением OTF. 
- Шрифты OpenType OTF могут быть установлены на любой версии Windows, а также на Mac OS 7.x-10.x.
- Шрифты OpenType базируются на кодировке Unicode, которая поддерживает практически все мировые языки.
- Шрифты OpenType можно инсталлировать средствами операционной системы или с помощью утилит управления 
   шрифтами, например, Adobe Type Manager.


Инструкции по установке шрифтов в различных ОС:

- - - - - -
Windows 7 / Windows Vista

Кликните правой кнопкой на файле шрифта и выберите Установить (Install) из контекстного меню.
Можно также просто скопировать шрифтовой файл в Панель управления Шрифты (Control Panel > Fonts) . 

-------
Windows XP

1. Откройте Панель управления (Control Panel). Кнопка Пуск (Start) > Панель управления (Control Panel).
2. Откройте (дважды щёлкните) панель Шрифты (Fonts). 
3. В окне Шрифты (Fonts) выберите Файл / Установить новый шрифт (File / Install New Font). 
4. Укажите папку, где лежат шрифты, которые бы вы хотите установить (шрифты должны быть распакованы из архивов). 
5. Выберите шрифты, которые вы хотите установить. Можете указать один шрифт или, удерживая клавишу Ctrl, несколько. 
6. Нажмите OK.  
7. Закройте окно Шрифты (Fonts). 
После инсталляции, если вы хотите, чтобы обновился список шрифтов в уже запущенных приложениях, вам нужно будет
перезапустить их.

Вы также можете установить шрифты просто скопировав шрифтовые файлы в Панель управления Шрифты 
(Control Panel > Fonts). 

- - - - - -
Mac OS X

Закройте все открытые приложения перед установкой шрифтов. 
Новые шрифты не появятся в некоторых приложениях, если эти приложения будут запущены в процессе установки шрифта. 

1. Откройте папку со шрифтами, которые вы хотите установить. 
2. Выберите файлы шрифтов, которые вы хотите установить. 
3. Удостоверьтесь, что вы выбрали шрифты, а не папки, содержащие шрифты.
3. Перетащите выбранные файлы в папку Fonts внутри папки Library на диске Mac OS X.

- - - - - -
Mac OS 7.x-9.x

Для шрифтов OpenType PS (OT/PS) во всех версиях Mac OS, за исключением версии Mac OS X, необходим Adobe Type Manager (ATM),
чтобы шрифты отображались более гладко на экране и при выводе на печать.
Для установки OT/PS шрифтов нужен ATM версии 4.6.2 или новее. 
Вы можете скачать свежую версию ATM Light бесплатно с сайта Adobe http://www.adobe.com/products/atmlight/main.html
